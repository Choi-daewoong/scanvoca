## 📋 출시 전 필수 작업

### 1. 프로덕션 빌드 테스트 ⚠️ 필수
**작업 내용:**
- EAS 빌드로 실제 APK/IPA 생성
- 빌드 에러 없는지 확인
- 앱 크기 확인 (100MB 이하 권장)

**명령어:**
```bash
cd app
eas build --platform android --profile production
eas build --platform ios --profile production
```

**예상 소요**: 1시간 (빌드 대기 포함)

---

### 2. 실제 기기 E2E 테스트 ⚠️ 필수
**테스트 시나리오:**
- [ ] 회원가입 → 로그인
- [ ] 갤러리에서 이미지 선택 → OCR → 단어 저장
- [ ] 단어장 생성 → 단어 추가 → 제목 변경
- [ ] 학습 모드 - 카드 뒤집기, 암기 표시
- [ ] 발음 재생 (🔊 버튼)
- [ ] 통계 화면 확인
- [ ] 설정 → 로그아웃

**예상 소요**: 1시간

---

### 3. 앱 아이콘 및 스플래시 스크린 ⚠️ 필수
**필요한 작업:**
- 앱 아이콘 디자인 (1024x1024 PNG)
- 스플래시 스크린 이미지
- app.json에 설정 추가

**파일 위치:**
- `app/assets/icon.png` (1024x1024)
- `app/assets/splash.png` (1284x2778 권장)

**예상 소요**: 30분 (디자인 제외)

---

### 4. 앱스토어 출시 자료 준비 ⚠️ 필수
**필요한 자료:**
- 앱 설명 (한글 200자 이내)
- 스크린샷 (최소 4장)
  - 홈 화면
  - 스캔 결과
  - 단어장 목록
  - 학습 모드
- 개인정보처리방침 URL (필수)
- 앱 카테고리: 교육

**예상 소요**: 2시간

---

### 5. 환경변수 보안 확인 ⚠️ 권장
**확인 사항:**
- [ ] OpenAI API 키가 코드에 하드코딩되지 않았는지
- [ ] .env 파일이 .gitignore에 포함되었는지
- [ ] 프로덕션 빌드 시 API 키가 난독화되는지

**예상 소요**: 30분
