# 사용자 시나리오 & UI/UX 설계

## 👥 타겟 사용자
- **주 사용자**: 중/고등학생 (13-18세)
- **보조 사용자**: 영어 학습자, 어학원 학생
- **사용 환경**: 교실, 집, 도서관, 통학길

---

## 📱 핵심 사용 시나리오

### 시나리오 1: 교과서 단어 스캔 학습
**상황**: 영어 수업 시간, 교과서에 모르는 단어가 있음
```
1. 앱 실행 → 카메라 화면
2. 교과서 페이지를 카메라로 촬영
3. OCR로 텍스트 추출, 단어들 인식
4. 모르는 단어들을 선택
5. 각 단어의 뜻과 예문 확인
6. "수업 단어장"에 추가
7. 나중에 복습 모드로 학습
```

### 시나리오 2: 시험 대비 단어장 만들기
**상황**: 중간고사 준비, 단어 시험 범위 정리
```
1. 단어장 화면 → "시험 대비" 새 단어장 생성
2. 여러 페이지/자료를 스캔하여 단어 수집
3. 중요도별로 단어 분류 (별표 표시)
4. 플래시카드 모드로 반복 학습
5. 틀린 단어들만 따로 복습
6. 학습 진도 확인
```

### 시나리오 3: 일상 영어 단어 수집
**상황**: 유튜브, 웹사이트, 소설에서 모르는 단어 발견
```
1. 빠른 스캔 모드로 단어 촬영
2. 즉시 뜻 확인
3. "일상 단어장"에 자동 저장
4. 주기적으로 복습 알림
5. 레벨별 학습 진행
```

### 시나리오 4: 그룹 스터디 활용
**상황**: 친구들과 함께 영어 공부
```
1. 공통 주제로 단어장 생성
2. 각자 발견한 단어들 추가
3. 서로 퀴즈 문제 내기
4. 학습 진도 비교
5. 성취도 공유
```

## 🎯 사용자 니즈 분석

### 기능적 니즈
1. **빠른 단어 인식**: 3초 이내 OCR 처리
2. **정확한 의미**: 교과서 수준의 정확한 한국어 뜻
3. **맥락 이해**: 실용적인 예문 제공
4. **체계적 관리**: 주제별/난이도별 단어장 분류
5. **효과적 학습**: 반복 학습 및 진도 추적

### 감정적 니즈
1. **성취감**: 학습 진도 시각화, 레벨업
2. **편리함**: 직관적 인터페이스, 최소 터치
3. **신뢰감**: 정확한 정보, 안정적 성능
4. **재미**: 게임화 요소, 다양한 학습 모드

### 상황적 니즈
1. **오프라인 학습**: 인터넷 없어도 모든 기능 사용
2. **배터리 효율**: 장시간 사용 가능
3. **빠른 접근**: 앱 실행 즉시 스캔 가능
4. **다양한 환경**: 밝기, 각도 등 다양한 촬영 조건

## 📋 주요 화면별 요구사항

### 1. 홈 화면
**목적**: 전체 학습 현황 파악 및 빠른 접근
```
- 오늘의 학습 목표 및 달성률
- 최근 추가한 단어들 미리보기
- 복습이 필요한 단어 개수
- 빠른 스캔 버튼 (플로팅)
- 학습 통계 그래프
```

### 2. 카메라/스캔 화면
**목적**: 효율적인 단어 스캔 및 즉시 학습
```
- 실시간 카메라 프리뷰
- OCR 영역 가이드라인
- 인식된 단어들 오버레이 표시
- 단어 선택 인터페이스
- 갤러리에서 이미지 선택 옵션
- 스캔 히스토리
```

### 3. 단어 상세 화면
**목적**: 단어의 완전한 이해
```
- 단어 + 발음기호
- 다양한 의미 (품사별)
- 실용적 예문 (2-3개)
- 난이도 표시 (CEFR 레벨)
- 관련 단어/동의어
- 단어장 추가 버튼
- 즐겨찾기 토글
```

### 4. 단어장 화면
**목적**: 체계적인 단어 관리
```
- 단어장 목록 (카드 형태)
- 각 단어장별 진도율
- 검색 및 필터링
- 정렬 옵션 (날짜, 이름, 진도)
- 새 단어장 생성
- 단어장 공유 기능
```

### 5. 학습 모드 화면
**목적**: 효과적인 암기 및 복습
```
- 플래시카드 인터페이스
- 진도바 (현재 위치)
- 정답/오답 버튼
- 힌트 기능
- 건너뛰기 옵션
- 학습 완료 피드백
```

### 6. 퀴즈 화면
**목적**: 학습 효과 검증
```
- 객관식/주관식 문제
- 제한시간 표시
- 정답률 실시간 표시
- 결과 화면 (점수, 틀린 문제)
- 약점 분석
- 재시도 옵션
```

### 7. 설정 화면
**목적**: 개인화 및 앱 관리
```
- 알림 설정
- 학습 목표 설정
- 데이터 관리 (백업/복원)
- 앱 정보
- 도움말 및 튜토리얼
```

## 🎨 디자인 원칙

### 1. 단순함 (Simplicity)
- 핵심 기능에 집중
- 불필요한 요소 제거
- 한 화면에 한 가지 주요 작업

### 2. 직관성 (Intuitiveness)
- 학습 없이 사용 가능한 인터페이스
- 일관된 아이콘과 버튼
- 예상 가능한 동작

### 3. 접근성 (Accessibility)
- 큰 터치 영역
- 높은 대비 색상
- 다양한 화면 크기 지원

### 4. 피드백 (Feedback)
- 즉각적인 반응
- 명확한 상태 표시
- 진행 상황 시각화
