## 🎯 인터랙션 디자인

### 제스처 및 터치
- **스와이프**: 카드 넘기기, 탭 전환
- **탭**: 선택, 실행
- **길게 누르기**: 추가 옵션 메뉴
- **핀치**: 확대/축소 (카메라)
- **더블탭**: 즐겨찾기 토글

### 애니메이션
- **페이지 전환**: 슬라이드 (300ms)
- **카드 뒤집기**: 3D 플립 (400ms)  
- **버튼 피드백**: 스케일 (150ms)
- **로딩**: 페이드 + 스피너
- **성공**: 체크마크 + 바운스

### 피드백
- **햅틱**: 성공/실패, 버튼 터치
- **사운드**: 정답/오답, 완료
- **시각적**: 색상 변화, 아이콘 변화
