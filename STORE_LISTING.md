# Scan Voca - 서비스 소개 / 마케팅 카피

> 이 문서는 과거 React Native 앱을 Google Play/App Store에 출시하기 위해 작성한 스토어 리스팅 문서였습니다.
> 현재 Scan Voca는 **웹 서비스**로 운영되므로, 앱 스토어 관련 항목(빌드, 스토어 등록, 스크린샷 규격 등)은 제거하고
> 랜딩 페이지·홍보용 카피로 재구성했습니다.

---

## 📚 서비스 이름
**Scan Voca - 스마트 영단어 학습**

---

## 🎯 한 줄 소개
```
사진으로 스캔하고, AI로 학습하는 스마트 영단어 웹 서비스
```

---

## 📖 전체 설명

### 한국어 버전

```
📚 Scan Voca - 사진 한 장으로 시작하는 스마트 영단어 학습

책, 노트, 교재의 영어 단어를 사진으로 찍기만 하면 자동으로 단어장에 저장!
AI가 뜻과 예문을 생성해주고, 다양한 학습 모드로 복습까지 완벽하게 지원합니다.

✨ 주요 기능

📷 스마트 스캔
• 카메라 또는 갤러리 이미지에서 영단어 자동 추출
• 분석할 영역을 직접 잘라서(크롭) 선택 가능
• 단어뿐 아니라 숙어·구동사(idiom/phrasal verb)도 인식

🤖 AI 기반 학습
• AI가 생성하는 정확한 한국어 뜻과 영어 정의
• 실생활 예문 자동 생성
• 단어별 난이도 자동 분류

📖 스마트 단어장
• 여러 개의 단어장을 폴더로 정리
• 단어장 공유 기능 (공유 코드로 친구에게 공유)
• 단어 뜻 직접 수정 가능
• 발음 듣기 (TTS)

🎯 다양한 학습 모드
• 플래시카드(Study), 객관식 퀴즈(Quiz), 시험(Exam), 철자 연습(Spelling)
• 학습 진행률 및 통계 대시보드

🔐 계정 및 동기화
• 이메일/비밀번호 또는 Google 로그인
• 클라우드 저장으로 어떤 기기에서든 이어서 학습

🎨 깔끔한 디자인
• 직관적인 UI, 다크모드 지원

🎓 이런 분들께 추천합니다
• 수능, 토익, 토플 준비 중인 학생
• 영어 원서나 자료를 읽는 직장인
• 효율적인 영단어 학습법을 찾는 모든 분

📞 문의 및 지원
• GitHub: https://github.com/Choi-daewoong/scanvoca

---

Scan Voca로 영단어 학습을 더 쉽고 재미있게 시작하세요!
```
