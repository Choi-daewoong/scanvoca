# Scan Voca 블로그 (콘텐츠 마케팅) 기획서

> 2026-07-14 브레인스토밍 결과 정리. 이 파일은 `scanvoca.com/blog` 구축 작업의 출발점 문서입니다.

## 1. 목적

- **Scan Voca(scanvoca.com) 홍보**가 최종 목적. 영어 학습 관련 콘텐츠를 발행해 검색 유입 → 서비스 전환을 노린다.
- 타겟 검색엔진: **구글 + 네이버** 둘 다.
- 글 원본은 유실/이사 걱정 없이 보관되어야 한다 (원래 "개인 홈페이지 백업" 아이디어의 진짜 요구사항).

## 2. 핵심 결정 사항

### 별도 개인 홈페이지는 만들지 않는다 → 블로그를 `scanvoca.com/blog`에 만든다
- 콘텐츠가 쌓아주는 SEO 가치(도메인 권위)를 제품 도메인에 직접 쌓기 위함.
- 방문자가 글을 읽은 뒤 클릭 한 번으로 제품에 도달.
- "백업"은 웹사이트가 아니라 **원본 = 마크다운 파일 = git 저장소**로 해결. 나중에 어디로 옮기든 마크다운을 다시 빌드하면 된다.
- 같은 글을 두 도메인에 올리면 구글 중복 콘텐츠 문제로 원본 선정을 통제할 수 없게 됨 → 원본(canonical)은 항상 scanvoca.com.

### 네이버/티스토리 완전 자동 포스팅은 불가능 (공식 API 종료)
- 네이버 블로그 글쓰기 API: **2020년 5월 종료** ([뉴스핌 기사](https://www.newspim.com/news/view/20200413000737)) — 외부 프로그램 자동 게재를 막기 위한 조치.
- 티스토리 오픈 API: 2024년 종료.
- 브라우저 자동화(Playwright 등) 우회는 **계정 저품질(검색 누락) 위험**이 커서 홍보용 블로그에는 금물.
- 네이버는 **유사문서 필터**가 있어 전문 복붙은 오히려 검색에서 밀림 → 네이버용 글은 "재작성된 도입부 + 핵심 요약 + 원문/서비스 링크" 형태가 유리.

## 3. 아키텍처 (이중 채널 전략)

| 채널 | 역할 | 자동화 수준 |
|---|---|---|
| `scanvoca.com/blog` | **원본(canonical)**, 구글 SEO 담당 | ✅ 완전 자동 |
| 네이버 블로그 | 네이버 통합검색(블로그 탭) 유입 담당 | △ 반자동 (붙여넣기 30초) |

```
옵시디언에서 글 작성 (frontmatter: title, description, tags, date, publish: true)
        │  git push (Obsidian Git 플러그인으로 자동 커밋/푸시)
        ▼
Scan_Voca 저장소 content/blog/*.md
        │  Vercel이 자동 감지 → 빌드/배포
        ├─▶ scanvoca.com/blog/[slug]  ← 원본 게시 (완전 자동)
        └─▶ 변환 스크립트: 네이버 붙여넣기용 버전 자동 생성
             (AI로 도입부 재작성 + 요약 + 링크 → 수동 붙여넣기만)
```

## 4. 만들 것 (작업 목록)

1. **`scanvoca.com/blog` 블로그 섹션** — 기존 Next.js 16 App Router 웹앱(`web/`)에 추가
   - `content/blog/*.md`(또는 MDX) 읽어서 정적 생성
   - 글 목록 페이지 + 글 상세 페이지
   - SEO: 메타태그(OG 포함), `sitemap.xml`, RSS 피드, JSON-LD(Article), canonical URL
2. **옵시디언 → git → 배포 연결**
   - 옵시디언 vault 안에 저장소의 `content/blog/` 폴더를 두거나 연결
   - Obsidian Git 플러그인으로 자동 커밋/푸시 설정
   - `publish: true`인 글만 빌드에 포함하는 필터
3. **네이버용 변환 스크립트**
   - 새 글 감지 → AI로 네이버용 버전 생성(유사문서 필터 회피를 위해 재작성) → 클립보드/파일로 출력
4. **검색엔진 등록** (코드 아님, 운영 작업)
   - 구글 서치 콘솔에 sitemap 제출
   - 네이버 서치어드바이저에 scanvoca.com 등록 (네이버 웹문서 검색 노출용)
